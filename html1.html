HTML

HTML la gi

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu dùng để tạo cấu trúc và nội dung cho trang web.

2. Cấu trúc cơ bản của một trang HTML

  • <!DOCTYPE html>: Khai báo đây là một tài liệu HTML5.
  • <html lang="vi">: Thẻ gốc chứa toàn bộ nội dung trang web (với lang="vi" để đặt ngôn ngữ tiếng Việt).
  • <head>: Chứa thông tin về trang web như tiêu đề, mã hóa ký tự, v.v.
  • <body>: Chứa nội dung chính của trang web.

3. Các thẻ HTML quan trọng

a) Tiêu đề & văn bản

  • <h1> đến <h6>: Tiêu đề từ lớn đến nhỏ.
  • <p>: Đoạn văn bản.
  • <br>: Xuống dòng
  • <hr>: Kẻ đường ngang.

b) Liên kết (Links)

  • <a href="URL">: Tạo liên kết đến trang khác.
  • <a href="URL" <target="_blank">>: Mở link trong tab mới.

c) Hình ảnh

  • <img src="URL" alt="Mô tả">: Hiển thị ảnh.
  • width & height: Điều chỉnh kích thước.

4. Cách chạy thử trang HTML

  1. Mở Notepad (hoặc VS Code, Sublime Text).
  2. Dán đoạn HTML vào.
  3. Lưu file với đuôi .html (ví dụ: index.html).
  4. Nhấp đúp vào file để mở trên trình duyệt.
